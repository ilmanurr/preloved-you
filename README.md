<br><p></p>Pertama-tama, kami mendeklarasikan tipe dokumen HTML dan memberitahu browser bahwa halaman ini adalah dokumen HTML 5 dengan menggunakan kode " DOCTYPE! html". Selanjutnya, kode "html lang=”en”" merupakan tag pembuka untuk elemen HTML dan menunjukkan bahwa bahasa yang digunakan dalam halaman ini adalah bahasa Inggris. Kemudian ada tag ‘head’ yang merupakan bagian kepala HTML yang berisi informasi meta, judul halaman dan tautan ke berkas eksternal. Disini kami menggunakan beberapa tag yaitu, tag ‘meta charset=”UTF-8”’ untuk mendefinisikan karakter set dokumen sebagai UTF-8 yang merupakan standar umum untuk karakter internasional, tag ‘meta http-equiv=”X-UA-Compatible” content=”IE-edge”’ untuk memberikana petunjuk kepada browser agar menggunakan mode terbaru atau standar terkini, tag ‘meta name=”viewport” content=”width=device-width, initial-scale=1.0” untuk mendefinisikan pengaturan tampilan untuk perangkat seluler dengan lebar layer yang sesuai dan tingkat zoom awal, tag ‘<tittle>Preloved-You</tittle>’ untuk menentukan judul halaman yang akan ditampilkan di tab browser, kemudian pada baris ke11-13 kami menggunakan tag-tag tersebut untuk mendefinisikan sambungan awal ke server font Google dengan memaksa penggunaan CORS (Cross-Origin Resource Sharing) lalu kami memasukkan gaya font Poppings dari Google Fonts. Selanjutnya terdapat tag script yang menghubungkan dengan library feather icon agar dalam website kami, kami dapat menggunakan icon yang ada dalam feather icon. Feather Icons adalah kumpulan ikon open source yang dapat diintegrasikan ke dalam proyek web. Kemudian, dengan tag ‘link rel=”stylesheet” href=”css/style.css”’ kami juga menautkan halaman index dengan file stylesheet eksternal yaitu ‘style.css’ yang digunakan untuk mendefinisikan dan mengatur gaya atau tampilkan halaman, ‘href=”css/style.css’ disini menunjukkan bahwa stylesheet css diambil dari file style.css berada di folder css dalam proyek yang sama. Perlu diketahui bahwa semua halaman yang ada dalam website saya memiliki tag html dan tag head yang isinya sama dengan halaman index ini.</p>
 <br><img src="gambar source code/html/index2 navbar.png">
 <br>Selanjutnya, kami menggunakan tag body yang merupakan bagian tubuh atau bagian utama dari dokumen HTML yang berisi konten-konten dalam halaman website. Kami menggunakan tag ‘nav class=”navbar” untuk mendefinisikan sebuah navigation bar dengan nama classnya adalah navbar. Kami membuat division dengan menggunakan tag ‘div’ yang memiliki class bernama “isinavbar”. Daftar item navigasi dalam class isinavbar dibuat menggunakan tag ‘ul’ dan ‘li’ yang mendefinisikan unordered list dalam HTML. Setiap item atau navigasi ini memiliki tag ‘a’ untuk menghubungkan antar halaman dengan halaman lainnya.
